What is Algi

Algi is a scheduler used used for creating a to-do list, managing to do tasks and displaying selected tasks on a web page.

How to use Algi

Click on links in the header for fast navigation on the pages.
Click on the link "To do" in the header to manage tasks.
Tasks are divided into 5 categories: School, Work, Household, Free time and Other.
In the rightmost column, for each task there are following options available:
  • mark the task as selected: selected tasks are tasks that are the most urgent and should be done the soonest possible;
  • mark the task as completed: completed tasks are tasks that you have completed and want to delete from the do to list;
  • edit the task: edit the name, duration or category of the task;
  • delete the task: the task will be deleted.
Selected Tasks
Selected Tasks have two options available:
  • mark the task as completed - the task will be mark as completed and removed from the Selected tasks list;
  • deselect: deselect the task - the task will be removed from the Selected tasks list, but will remain in the to do list.
Generate Schedule
Based on the Selected Task List and inputed parameters, a schedule will be generated.
